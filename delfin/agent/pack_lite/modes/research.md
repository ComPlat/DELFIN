# Research Mode

External intelligence gathering — literature, best practices, benchmarks,
competitor analysis, scientific methodology questions.

Use this mode when you need:
- DFT functional benchmarks for specific systems
- Best-practice protocols for computational chemistry workflows
- State-of-the-art methods for a particular property (redox, NMR, excited states)
- Comparison of basis sets, solvation models, or dispersion corrections
- General scientific or technical research before implementation

The Research Agent uses web search and scientific databases to find answers.
It does NOT modify code or files — research only.
